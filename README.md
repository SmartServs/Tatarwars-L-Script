# 🏹 Tatarwars-L Script (Lite Version)

سكربت **حرب التتار - النسخة الخفيفة (L-V1 2025)**  
مصمم لعشاق اللعب الكلاسيكي البسيط والسريع بدون تعقيدات أو إضافات مملة.

---

## 📌 عن المشروع

سكربت **Tatarwars-L** هو نسخة Lite (خفيفة) من لعبة حرب التتار،  
موجهة للاعبين اللي بيدوروا على:

- ⚡ سرعة تصفح عالية جدًا  
- 🎮 تجربة لعب كلاسيكية (شبيهة بترافيان القديمة)  
- 🧠 تحدي حقيقي بدون تسهيلات  
- 🚫 بدون إضافات أو مزايا تخلي اللعبة سهلة أو مملة  

---

## 🚀 المميزات

- 🪶 سكربت خفيف وسريع  
- 🧩 بسيط وسهل التعديل  
- ⚔️ نظام لعب متوازن وصعب  
- 💻 مناسب للسيرفرات الضعيفة  
- 🔒 بدون تعقيدات أو أنظمة حديثة مزعجة  

---

## ⚙️ متطلبات التشغيل

- PHP 5.6  
- MySQL Database  
- Web Server (Apache أو Nginx)  

---

## 🛠️ طريقة التركيب

### 1. تعديل الإعدادات

افتح الملف التالي:
core-f/config-f/s1.php

قم بتعديل بيانات الاتصال بقاعدة البيانات:
- DB_HOST  
- DB_USER  
- DB_PASS  
- DB_NAME  

---

### 2. رفع الملفات

قم برفع جميع ملفات السكربت على السيرفر.

---

### 3. تشغيل التثبيت

افتح الرابط التالي في المتصفح:

http://www.yoursite.com/login.php?myinstall=install9

---

### 4. ملاحظة

يمكنك تغيير رابط التنصيب من خلال ملف الكونفيج حسب رغبتك.

---

## ⚠️ ملاحظات مهمة

- تأكد من ضبط إعدادات قاعدة البيانات بشكل صحيح  
- لا تشارك بيانات الاتصال (Database Credentials)  
- يفضل تشغيل السكربت على PHP 5.6
- اذا ظهر لك خطأ 503 عند التنصيب تأكد من حذف ملف install.txt الموجود داخل فولدر core-f\stx
